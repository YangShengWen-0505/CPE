# 🧮 Concept: Complex Number Rotation (Niven's Theorem)

> **核心目標**：給定一個複數 $z = a + bi$，尋找最小的正整數 $n$，使得 $z^n$ 成為一個**純實數**（虛部為 0）。

---

## 📐 數學公式與幾何定義

### 1. 複數的極式 (Polar Form)
任何複數 $z = a + bi$ 都可以表示為長度（模長）與角度（輻角）的形式：
$$z = r(\cos \theta + i \sin \theta)$$
* **模長** $r = \sqrt{a^2 + b^2}$
* **輻角** $\theta = \arctan(\frac{b}{a})$

### 2. 棣美弗定理 (De Moivre's Theorem)
當複數進行 $n$ 次方運算時，其幾何意義為「模長變為 $r^n$，角度旋轉 $n$ 倍」：
$$z^n = r^n(\cos(n\theta) + i \sin(n\theta))$$



---

## 🧠 核心邏輯推導

### 成為「實數」的幾何條件
若要讓 $z^n$ 成為實數，代表其在複數平面上旋轉後必須落在 **X 軸（實數軸）** 上。
這意味著虛部必須為零：
$$\sin(n\theta) = 0 \implies n\theta = k\pi \quad (k \in \mathbb{Z})$$

### 尼雲定理 (Niven's Theorem) 的限制
在多數程式競賽中，$a$ 與 $b$ 為整數。這代表 $\tan \theta = \frac{b}{a}$ 是一個**有理數**。
根據尼雲定理，若 $\theta$ 是 $\pi$ 的有理數倍，且 $\tan \theta$ 也是有理數，則 $\tan \theta$ 的值只可能是 $0, \pm 1$ 或無定義（分母為 0）。

這說明了只有特定的角度可以透過整數次方旋轉回到實數軸，其他的角度無論旋轉幾次都無法精確落在軸線上。

---

## ⚡ 決策邏輯 (Decision Logic)

根據上述原理，我們可以將判斷邏輯簡化為以下四種情況，這也是複習時最重要的判斷準則：

| 判斷條件 | 角度狀態 | 最小 $n$ 值 |
| :--- | :--- | :--- |
| **$b = 0$** | 點已在實數軸上 | **1** |
| **$a = 0$** | 點在虛數軸 ($90^\circ$ 或 $270^\circ$) | **2** |
| **$|a| = |b|$** | 點在角平分線上 ($45^\circ$ 的倍數) | **4** |
| **其餘情況** | 無理數旋轉角，永遠無法旋轉至軸線 | **TOO COMPLICATED** |

---

## 🛠️ 語法與實作注意點 (Syntax Tips)

1.  **絕對值判斷**：
    * 在 C++ 中判斷 $|a| = |b|$，請使用 `abs(a) == abs(b)`。
    * 需引入標頭檔 `<cmath>` 或 `<cstdlib>`。
2.  **型別注意**：
    * 即便最終邏輯是 O(1)，但輸入的 $a, b$ 可能很大（例如 $2^{31}-1$），建議使用 `long long` 接收輸入以避免溢位風險。
3.  **邏輯優先級**：
    * 必須優先檢查 $b=0$，再檢查 $a=0$，最後才是 $|a|=|b|$。
